# Stock Study: 이동평균선 대순환 분석 시스템

이 프로젝트는 일본의 투자 교육가 고지로 강사의 **'이동평균선 대순환 분석'** 이론을 기반으로 주식 데이터를 수집, 분석 및 시뮬레이션하는 시스템입니다.

## 1. 핵심 투자 전략
- **이동평균선 대순환**: 5일, 20일, 40일 지수이동평균(EMA)의 배열에 따라 시장을 6개 스테이지로 구분합니다.
  - **Stage 1 (정배열)**: 5 > 20 > 40 (안정적 상승 국면 - 매수/보유)
  - **Stage 4 (역배열)**: 40 > 20 > 5 (안정적 하락 국면 - 매도/관망)
- **대순환 MACD**: 이동평균선보다 선행하는 지표로, 단기(5-20), 중기(5-40), 장기(20-40) EMA 차이를 분석합니다.
- **Early Buy (선발대)**: Stage 5/6 상태에서 3개의 MACD 지표가 모두 상향 반전할 때 조기 매수하는 전략입니다.

## 2. 주요 파일 및 역할

### 데이터 수집 및 업데이트
- **`fetch_stock_data.py`**: 한국 시장(KOSPI, KOSDAQ) 전 종목 데이터를 초기 수집합니다.
- **`fetch_us_stock_data.py`**: 미국 시장(NASDAQ, NYSE, AMEX) 주요 종목 데이터를 수집합니다.
- **`incremental_update.py`**: **(핵심)** DB의 마지막 날짜 이후 데이터만 멀티스레딩(10개 스레드)을 통해 효율적으로 업데이트합니다.
- **`fetch_qqq.py`**: QQQ 등 특정 종목의 데이터를 별도로 업데이트합니다.

### 종목 스캔 및 분석
- **`scan_buy_timing.py`**: 오늘 막 Stage 1에 진입한 종목을 포착하고 차트를 생성합니다.
- **`scan_early_buy.py`**: **(선발대)** Stage 5/6에서 MACD 반등 신호가 포착된 종목을 찾아 `plot_macd_XXXXXX.png` 차트를 생성합니다.
- **`visualize_kojiro.py`**: 캔들 차트, 이동평균선, 대순환 MACD를 포함한 고해상도 분석 이미지를 생성합니다.


## 3. 사용 방법

### 데이터 업데이트 (매일 장 마감 후)
```bash
uv run python incremental_update.py
```

### 오늘 자 매수 신호 종목 확인
```bash
# 선발대(Early Buy) 신호 및 차트 생성
uv run python scan_early_buy.py
```

### 특정 종목 상세 분석 차트 생성
```bash
uv run python visualize_kojiro.py 005930  # 삼성전자 차트 생성
```

## 4. 백테스트 결과 요약 (Kojiro Early Buy 전략)
- **성공 확률**: 대형주 및 지수 ETF 기준 **약 75% ~ 85%**가 신호 발생 후 Stage 1 진입에 성공.
- **리스크 관리**: 평균 손실폭을 **-2% 내외**로 제한하며, 추세 발생 시 최대 수백%의 수익을 추구.
- **방어력**: 단순 보유(Buy & Hold) 시 겪어야 하는 **-70~90%의 MDD**를 효과적으로 회피.
